# 🇪🇨 Bienvenido a MCP Hub Ecuador

¡Hola! 👋 Somos la iniciativa de código abierto dedicada a estandarizar y facilitar la interoperabilidad entre **Agentes de Inteligencia Artificial** (Claude, OpenClaw, Copilot, Ollama) y el ecosistema tecnológico, financiero y tributario de Ecuador.

## 🎯 Nuestra Misión

Proveer conectores basados en el **Model Context Protocol (MCP)** que sean seguros, auditables, _stateless_ y listos para producción. Queremos que cualquier agente IA pueda facturar, consultar y cobrar en Ecuador de forma nativa en cuestión de minutos.

## 🧭 ¿Por dónde empiezo?

Si eres nuevo aquí, tu primer paso debe ser visitar nuestro repositorio central (**Hub**). Ahí encontrarás nuestro catálogo oficial, reglas de arquitectura y guías de contribución:

👉 [**Visita el Directorio Central (Hub) aquí**](https://github.com/mcphub-ec/hub "null") 👈

> 🤖 **¿Eres un Agente IA?** Por favor, lee inmediatamente nuestro [Agent Cheatsheet](https://github.com/mcphub-ec/hub/blob/main/agent-cheatsheet.md "null") para aprender las reglas de negocio de Ecuador (ej. 15% de IVA, formato de RUC/Cédulas, etc.).

## 🛠️ Tecnologías Soportadas (Catálogo Destacado)

Mantenemos repositorios individuales para cada integración, asegurando modularidad y despliegues limpios (vía Docker):

### 🏛️ SRI y Contabilidad

-   [**mcp-sri**](https://github.com/mcphub-ec/mcp-sri "null")**:** Emisión nativa, firma XAdES-BES y transmisión directa al SRI.
-   [**mcp-contifico**](https://github.com/mcphub-ec/mcp-contifico "null")**:** Conector integral para el ERP Contífico.
-   [**mcp-invoka**](https://github.com/mcphub-ec/mcp-invoka "null")**:** Facturación simplificada basada en créditos.

### 💳 Pasarelas de Pago

-   [**mcp-payphone**](https://github.com/mcphub-ec/mcp-payphone "null")**:** Generación de links y cobros Push al instante.
-   [**mcp-kushki**](https://github.com/mcphub-ec/mcp-kushki "null")**:** Tokenización y procesamiento directo de tarjetas.
-   [**mcp-pagomedios**](https://github.com/mcphub-ec/mcp-pagomedios "null")**:** Ecosistema Abitmedia para automatizar flujos de pago.
